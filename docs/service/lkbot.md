# lk插件

## 服务信息
- **服务名**：lk插件
- **说明**：l_o_o_k的综合性插件
- **限定权限**：无

## 命令列表
- **签到**：全新界面的签到系统
- **我的信息**：查询自己的信息
- **我的背包**：查看背包中的内容
- **物品查询**：查询指定物品信息
- **/使用**：使用指定物品,'全部物品'使用全部,'物品*n'使用n个物品
- **/回收**：将指定数量物品换成ATRI币,'全部物品'使用全部,'物品*n'回收n个物品
- **商店列表**：列出所有商店
- **商品列表**：列出指定商店的商品列表
- **/购买**：从指定商店中购买指定数量的商品
- **/lk.改名**：用改名卡修改自己的名称
- **/lk.新内容**：展示更新内容
- **/lk.绑定**：为自己绑定一个名称
